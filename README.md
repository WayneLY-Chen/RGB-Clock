# RGB-Clock
"A desktop digital clock application built with Python and Tkinter. It features a smooth, dynamic RGB color cycling effect applied to the text. The project utilizes the colorsys library to convert HSV values to RGB, enabling a fluid rainbow gradient animation that updates in real-time."
